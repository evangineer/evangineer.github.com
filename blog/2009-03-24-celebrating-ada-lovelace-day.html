Celebrating Ada Lovelace Day

Today is Ada Lovelace Day so I would like to take the opportunity to highlight a woman in technology I admire Leah Culver. Leah developed the now defunct Pownce social network. She not only built Pownce using Django but is active on the Data Portability front having developed and opensourced django-elsewhere. She’s also helping create the Open Web by working on a Python OAuth library & OEmbed. I’m sure even now she’s cranking useful code for interesting projects. Respect.
